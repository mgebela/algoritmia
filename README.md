# Algoritmija

**Romeo i Julija: Algoritmi i roboti u umjetnosti**

Edukativna radionica koja spaja kazališnu umjetnost i tehnologiju. Polaznici programiraju robote s ljudskim izrazima lica i zvukom kako bi postavili scene iz predstave „Romeo i Julija".

## Pokretanje lokalno

Otvorite `index.html` u pregledniku ili pokrenite lokalni server:

```bash
cd ~/Downloads/algoritmia
python3 -m http.server 8080
```

Zatim posjetite [http://localhost:8080](http://localhost:8080).

## GitHub Pages

Repozitorij je spreman za objavu putem GitHub Pages — u postavkama repozitorija odaberite **Source: Deploy from branch → main → / (root)**.

## Struktura

```
algoritmia/
├── index.html          # Glavna stranica
├── css/style.css       # Stilovi
├── js/main.js          # Galerija, lightbox, animacije
└── images/gallery/     # Optimizirane fotografije radionice
```
